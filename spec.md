# Batalha Naval Web

MVP multiplayer para dois jogadores via browser.

- Frontend: React, TypeScript e Vite.
- Backend: Node.js, TypeScript, Fastify e Socket.IO.
- Regras compartilhadas: pacote `@batalha-naval/shared`.
- Estado em memoria no backend.
- Sem login, banco de dados ou camada de seguranca nesta fase.
